Cette application ajoute les informations des autobus de Cornwall Transit à MonTransit.

L'app contient les horaires planifiés.

Les autobus de Cornwall Transit desservent Cornwall en Ontario, Canada.

Une fois cette application installée, l'application MonTransit affichera les informations des autobus (horaires ...).

Cette application a seulement une icône temporaire : télécharger l'app MonTransit (gratuit) dans la section "Autres ..." ci-dessous ou en cliquant sur ce lien Google Play https://goo.gl/pCk5mV

Vous pouvez installer cette application sur la carte SD mais ce n'est pas recommandé.

Ces horaires sont générées à partir du fichier GTFS fourni par la Cornwall Transit.
https://www.cornwall.ca/fr/city-hall/open-data.aspx

Cette application est gratuite et open-source : 
https://github.com/mtransitapps/ca-cornwall-transit-bus-android

Cette application n'est pas associée à Cornwall Transit.
